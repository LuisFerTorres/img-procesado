# Procesamiento de Imagenes

Para obtener las imagenes transformadas ejecutar `main.py`, esto crea las carpetas

* `img_grayscale`: las imagenes en escala de grises y sus histogramas
* `img_tr`: las imagenes transformadas por las funciones en `transformaciones.py` y sus histogramas
* `img_fun`: el comportamiento del LUT bajo las transformaciones

El codigo es dependiente de las siguientes librerias:

* matplotlib
* numpy
* opencv-python
